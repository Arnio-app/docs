# Navigating the Dashboard

## <Icon icon="message" />  Conversation Tab

Track and manage all leads conversations in real time.

- View incoming and outgoing messages
- Filter by lead status, rep or time frame
- Click into a thread to view message history

![Hero Dark Pn](/images/hero-dark.png)

## <Icon icon="gear-complex" />  Automation Tab

Set up and manage sequences, message triggers, and workflows.

•	Create multi-step automations (email \+ SMS \+ task reminders)
•	Use filters to assign automations based on lead attributes
•	Monitor performance of each automation

![Hero Dark Pn](/images/hero-dark.png)

## <Icon icon="toolbox" />  Settings Tab

Configure global account and user preferences.

•	Manage users and permissions
•	Set fallback reps
•	Add integrations (HubSpot, Salesforce, etc.)

![Hero Dark Pn](/images/hero-dark.png)

## <Icon icon="chart-line" />  Usage Tab

Review team performance and platform usage.
•	Response rate (split by iMessage and SMS)
•	Total and average response time
•	Pickup rate and unresponded message metrics

![Hero Dark Pn](/images/hero-dark.png)
